# Willow Daycare Website

A modern, responsive website for Willow Daycare - a family-run nursery in Carmarthen, Wales.

## About

Willow Daycare is a family-run business that puts children at the heart of our care. Located on Glangwili Hospital grounds in Carmarthen, we provide exceptional childcare services with a focus on creating a home-from-home environment.

## Features

- **Responsive Design**: Works beautifully on all devices
- **Multiple Pages**: Home, About, Services, Gallery, Jobs, Fundraising, and Contact
- **Modern UI**: Clean, professional design matching the Willow Daycare brand
- **Image Integration**: Uses provided logo, header background, certificate, job post, and fundraising images
- **Social Media Links**: Direct links to Facebook and Instagram
- **Contact Information**: Easy access to phone, email, and location details

## Technology Stack

- **Next.js 14**: React framework with App Router
- **TypeScript**: Type-safe development
- **CSS Modules**: Scoped styling for components
- **Responsive Design**: Mobile-first approach

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Building for Production

```bash
npm run build
npm start
```
